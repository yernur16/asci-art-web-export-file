Ascii-art-stylize consists in creating an attractive, interactive and intuitive website. Ascii-art-web consists in creating and launching a server on which a web GUI (graphical user interface) can be used On the main page you can see text input and 3 buttons that send a Post request to ‘Ascii-art’ and output the result to the page. Ascii-art is a program that consists in receiving a string as an argument and outputting a string in a graphical representation using ASCII. This project should handle input data with numbers, letters, spaces and special characters.
